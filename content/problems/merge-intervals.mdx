•	Asked to merge, combine, or remove overlaps. 
•	Need the minimum number of merged intervals.

## Core Intuition

Core Intuition
Two intervals overlap if the next interval starts before the current interval ends.
After sorting by start time:
Current : [a, b]
Next    : [c, d]

Overlap if:
c <= b
Merge them into:
[a, max(b, d)]

________________________________________
Why Sort?
Without sorting:
[5,7]
[1,6]
it's difficult to determine overlap consistently.
After sorting:
[1,6]
[5,7]
the condition becomes very simple:
next.start <= current.end


Merge Formula
Current = [a, b]
Next    = [c, d]

Overlap:
c <= b

Merged:
[a, max(b, d)]
________________________________________

## Algorithm

1.	Sort intervals by start time. 
2.	Initialize current = first interval. 
3.	Iterate through remaining intervals: 
o	If overlapping: 
	Extend current interval. 
o	Else: 
	Current interval is complete. 
	Add it to the answer. 
	Start a new current interval. 
4.	Add the last current interval.

•	Forgetting to sort first. 
•	Merging as [a, d] instead of [a, max(b, d)].

## Interview Mantra

Sort by start time → Maintain one current interval → Extend while overlapping → Finalize when a gap appears.

## Notes


Why Greedy Works
After sorting:
•	If next.start > current.end, then every future interval also starts after current.end. 
•	Therefore, the current interval can never merge again. 
•	It is safe to finalize it immediately. 
This is the greedy decision.

## Code (Java)

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        // sort the intervals by start; [start,end]
        Arrays.sort(intervals, (a,b) -> Integer.compare(a[0],b[0]));
        ArrayList<int[]> ans = new ArrayList<>();

        int[] current = intervals[0];
        for(int i=1; i<intervals.length; i++){
            int[] next = intervals[i];
            if(isOverlapping(current,next)){
                current[1] = Math.max(current[1],next[1]);
            }else{
                ans.add(current);
                current = next;
            };
        }
        ans.add(current);
    return ans.toArray(new int[0][]);
    }
    private boolean isOverlapping(int curr[], int next[]){
        int a = curr[0];
        int b = curr[1];
        int c = next[0];
        int d = next[1];
     return (a<=c && c<=b);
    }
}
```
