## Recognition

The question asks for a longest contiguous region while enforcing a validity rule. This is a sliding-window problem.

## Core Intuition

Keep a window that is always valid. Expand its right edge; when a duplicate appears, shrink from the left until validity returns.

## Algorithm

Use a set and two pointers. Every character enters and leaves the window at most once.

```python
def lengthOfLongestSubstring(s: str) -> int:
    seen, left, best = set(), 0, 0
    for right, char in enumerate(s):
        while char in seen:
            seen.remove(s[left])
            left += 1
        seen.add(char)
        best = max(best, right - left + 1)
    return best
```

## Complexity

Time O(n), space O(min(n, alphabet size)).

## Variants

Longest substring with at most K distinct characters; minimum window substring.
